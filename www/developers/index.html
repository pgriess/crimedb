CrimeDB - Developers

CrimeDB is intended to be an open repository of data for anyone to use. As such, all code is available on GitHub and licensed under ASL 2.0. All data is licensed under CC BY license.

All data is available at data.crimedb.org.

A list of all datasets available can be found by issuing a GET requset to http://data.crimedb.org/. This will return a JSON object describing the list of all valid datasets (regions).

Individual datasets are hosted in subdirectories. For example, http://data.crimedb.org/stl/ hosts the dataset for St. Louis City, MO. Issuing a GET request for this path will return metadata about this specific dataset in JSON format.

Datasets are broken down into files for each month of data collected. For example, http://data.crimedb.org/stl/2014-01.json contains data for all crimes reported to have occurred in January of 2014 in St. Louis City, MO. This meat of this JSON object is a crimes array, with each of the elements containing the following fields:

  • description -- freeform textual description of the crime
  • time -- time the crime occurred, in UTC epoch seconds
  • location -- a GeoJSON entity describing the location of the crime
